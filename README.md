## Getting started

### Installing

1. Clone the repo:

```bash
Git clone https://github.com/bjornhaavard/BidHub
```

2. install dependencies

```
npm i
npm i bootstrap
npm i sass
npm i prettier
```

3. Start sass watch and live server:
```
npm run start
```

4. build sass:
```
npm run build
```

5. format with prettier:
```
npm run format

```
